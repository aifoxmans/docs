---
title: "术语表"
---

文档中使用的关键术语定义。

| 术语             | **英文全称/缩写**                       | **解释说明**                                                    |
| :------------: | --------------------------------- | ----------------------------------------------------------- |
| **区块链**        | Blockchain                        | 一种去中心化的分布式数字账本技术，以时间顺序将交易数据记录在按顺序链接的区块中，确保其不可篡改和透明。         |
| **加密货币**       | Cryptocurrency                    | 基于密码学和区块链技术创建的数字或虚拟货币，用作交换媒介，例如比特币（BTC）、以太坊（ETH）。           |
| **稳定币**        | Stablecoin                        | 一种其价值与法定货币（如美元、欧元）或其他稳定资产挂钩的加密货币，旨在减少价格波动，例如 USDT, USDC。    |
| **智能合约**       | Smart Contract                    | 一种存储在区块链上的自执行合同，当预设条件被满足时，合约条款会自动执行，无需中介。                   |
| **法币**         | Fiat Currency                     | 由政府发行但不以实物商品（如黄金）为支撑的货币，其价值基于信用和法律规定，例如人民币（CNY）、美元（USD）。    |
| **商户**         | Merchant                          | 接受加密货币或稳定币作为其商品或服务支付方式的企业或个人。                               |
| **API**        | Application Programming Interface | 应用程序编程接口，允许不同的软件系统相互通信。支付平台通过API提供接口，让商户的系统能调用支付、查询等功能。     |
| **Key/Secret** | API Key & Secret                  | 平台为商户生成的一对密钥，用于身份验证和API请求签名，Key是公开的身份标识，Secret是必须保密的私钥。     |
| **Webhook**    | Webhook                           | 一种由事件触发的HTTP回调（或API）。支付平台通过Webhook主动向商户的服务器发送实时通知，例如支付状态更新。 |
| **回调地址**       | Callback URL                      | 由商户提供并预先配置在平台的一个URL地址，用于接收Webhook通知。                        |
| **Gas费**       | Gas Fee                           | 在区块链网络上执行交易或智能合约所需支付的计算费用，通常以该区块链的原生代币支付（如ETH）。             |
| **交易哈希**       | Transaction Hash/Txid             | 一串唯一标识区块链上一笔交易的字母数字字符串，用于查询和验证交易详情。                         |
| **区块确认**       | Block Confirmation                | 指一笔交易被打包进区块后，后续又有新的区块叠加在其之上。确认数越多，交易被逆转的可能性就越低，安全性越高。       |
| **结算**         | Settlement                        | 将成功收到的支付款项，扣除手续费后，记入商户账户余额的过程。                              |
| **提现/出金**      | Withdrawal                        | 将平台账户中的加密货币余额提取到外部区块链地址的过程。                                 |
| **OTC**        | Over-The-Counter                  | 场外交易。平台提供的内部撮合服务，允许用户（商户）直接与其他用户以协商好的价格大宗买卖加密货币，避免滑点。       |
| **KYC**        | Know Your Customer                | 了解你的客户。监管要求的身份验证流程，平台需收集和验证用户（商户）的身份信息，以预防欺诈和非法活动。          |
| **AML**        | Anti-Money Laundering             | 反洗钱。一系列旨在防止通过金融系统隐匿非法所得资金来源的法律法规和程序。                        |
| **冷钱包**        | Cold Wallet                       | 一种离线存储加密货币的钱包，私钥不接触互联网，安全性极高，通常用于存储大量资金。                    |
| **热钱包**        | Hot Wallet                        | 一种连接互联网的在线钱包，便于快速存取资金，但相对于冷钱包安全性较低。                         |
| **汇率浮动**       | Exchange Rate Fluctuation         | 加密货币相对于法币或其他加密货币的价格波动。支付平台通常为商户提供锁价功能来规避此风险。                |
| **终局性**        | Finality                          | 在区块链上，一笔交易一旦被确认即不可逆转、不可更改的状态。不同区块链的终局性时间不同。                 |
