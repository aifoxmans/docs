---
title: "快速开始"
---

#### **1. 提交资料与审核开户**

请准备并提交所需的商户资质文件，平台将在收到资料后尽快完成审核，并为审核通过的商户开通账户。

#### **2. 创建应用并获取凭证**

成功开户后，请登录商户后台创建应用，系统将为您生成唯一的 API Key 和 Secret Key。这些凭证是调用平台接口的重要身份标识，请妥善保管。

#### **3. 系统对接与流程配置**

技术人员可基于平台提供的API接口、Webhook 通知及 Callback 机制，将您的业务系统与平台的支付、结算等功能进行对接。如暂无业务系统，可跳过本步骤，直接使用商户后台进行相关操作。

#### **4. 发起支付与用户付款**

您可通过调用API或直接在商户后台创建支付订单，生成对应的收银台链接或页面。终端用户可通过该链接完成付款操作。

#### **5. 资金到账与提现处理**

资金到账后，您可随时将账户余额中的币种通过结算功能进行提现，也可通过平台内嵌的OTC服务出金为法币或其他数字货币。
